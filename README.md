# Orbital Archive

An interactive 3D portfolio built with React Three Fiber. Features an animated globe with clickable hotspots, smooth camera transitions, and a dark space aesthetic.

## Tech Stack

- **React** + **Vite**
- **Three.js** via React Three Fiber
- **GSAP** for camera animations and transitions
- **Zustand** for state management
- **Tailwind CSS** for UI styling
- **@react-three/postprocessing** for bloom and vignette effects

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |

## Features

- Interactive 3D globe with orbital particles and energy rings
- Clickable hotspots that navigate to portfolio sections
- GSAP-powered camera transitions between sections
- Mouse-based camera sway
- Post-processing: bloom and vignette
- Sections: About, Experience, Skills, Projects, Semester, Contact
